# Intelligent Door Access Control System Using Fuzzy Logic

## Overview

A Mamdani Fuzzy Inference System (FIS) developed to determine door access eligibility based on body temperature and mask compliance.

The system uses fuzzy logic to handle uncertain conditions and generate more flexible access decisions compared to traditional threshold-based approaches.

## Features

- Body Temperature Assessment
- Mask Compliance Evaluation
- Mamdani Fuzzy Inference System
- Centroid Defuzzification
- Rule-Based Decision Making

## Tools Used

- MATLAB
- Fuzzy Logic Toolbox
